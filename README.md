# Cloze Dropdown

Transform traditional Anki cloze cards into interactive dropdown quizzes.

Cloze Dropdown replaces standard cloze blanks with configurable dropdown menus, creating a clean multiple-choice learning experience while preserving the flexibility of normal cloze cards.

Originally built for my own daily study sessions and later shared with the Anki community.

# Features

- Interactive dropdown cloze cards
- Multiple-choice review
- Up to **9 answer options** per blank
- Alphabetical or random answer order
- Green / red answer feedback
- Preserves original text colours
- Optional Auto Flip
- Keyboard shortcuts
- Centered review layout
- White content container option
- Light & Dark mode support

# How It Works

1. Create a Cloze Dropdown card.
2. Review your card.
3. Choose answers from the dropdowns.
4. Flip the card or let Auto Flip do it automatically.
5. Continue reviewing as normal.

## Manual Installation

1. Download this repository.
2. Extract the files.
3. Copy the add-on folder into:

### Windows

```text
C:\Users\YOUR_NAME\AppData\Roaming\Anki2\addons21\
```

### macOS

```text
~/Library/Application Support/Anki2/addons21/
```

### Linux

```text
~/.local/share/Anki2/addons21/
```

4. Restart Anki.
